# MonQuest (ポケモン風ブラウザゲーム)

シンプルなポケモン風ゲームです。草むらを移動するとランダムエンカウントし、
攻撃・捕獲・逃走のターン制バトルを行えます。

## 起動方法

```bash
python -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いてください。

## 実装した要素

- タイルベースのフィールド移動（矢印 / WASD）
- 草むらエンカウント
- 4種の野生モンスター
- 2技 + 捕獲 + 逃走のバトルコマンド
- 捕獲による手持ち追加（最大6）
- レベルアップと全滅時の自動リカバリー
